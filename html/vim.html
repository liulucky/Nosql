vim之ctags插件

作者：梁芝 • 2016年6月15日

ctags是vim下方便代码阅读的工具。它可以让我们快速找到函数、类的所在位置。使用方法：使用鼠标点击变量名或函数名、类名，然后点击，这样就能直接跳到这个变量或函数定义的源文件中。
当然，最方便的方法是将光标移动到变量名或函数名上，然后按下"ctrl+]"，这样也可以直接跳到变量的源文件中。

安装方法： 

下载地址： http://ctags.sourceforge.net/

下载对应系统的包：

如果是window系统，解压收，找到ctags.exe复制到C:\Windows\System32目录下，这样在任何目录下都可以调用ctags这个命令。

然后在你的代码目录下运行命令ctags -R 生成tags文件，如图(我的代码是在E:\phpstudy\WWW\tpTest下的)

然后就可以在你的代码目录下看到tags文件了，接着修改我们的vim的配置文件_vimrc中添加关于ctags的设置： 

map  :!ctags -R --c++-kinds=+p --fields=+iaS --extra=+q .
imap  :!ctags -R --c++-kinds=+p --fields=+iaS --extra=+q .
set tags=E:\phpstudy\WWW\tpTest
set autochdir

到此，ctags安装成功，注意当你代码的目录发生改变，比如新增了文件我们需要再次运行

ctags -R
